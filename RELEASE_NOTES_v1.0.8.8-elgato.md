# v1.0.8.8 — Elgato Stream Deck

DeckScout for **Elgato Stream Deck** now supports both **Nightscout** and **Dexcom Share** in the same working release.

## Highlights

- ✅ Nightscout support preserved on the verified old Elgato baseline
- ✅ Direct **Dexcom Share** support added to the Elgato build
- ✅ Property Inspector settings now save reliably
- ✅ Key rendering fixed so live glucose data actually displays on the key
- ✅ Works on the official Stream Deck app with the old-compatible Elgato runtime shape

## Installation

1. Download `DeckScout-1.0.8.8-elgato.streamDeckPlugin` from this release
2. Open the file to install it in Stream Deck
3. Drag **DeckScout → Glucose Monitor** onto a key
4. Choose **Nightscout** or **Dexcom Share**
5. Save settings

## Notes

- Dexcom Share requires publisher credentials, not follower credentials
- Dexcom Share also requires at least one follower configured in Dexcom Share
- Phone-number usernames must include country code, e.g. `+11234567890`
- This is not medical advice and should not be used for treatment decisions
